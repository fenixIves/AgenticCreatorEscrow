## 赛道1: cobo 赛道｜agentic economy × cobo agentic wallet

Cobo 赛道关注 Agentic Commerce：构建让 AI 参与经济活动的应用或基础设施。参赛项目可以是一个能花钱、赚钱、交易的 Agent，也可以是让 Agent 经济运转的底层工具。

- **赛道建议方向（可选）**
- 01｜Agent-Native Payments

让 Agent 成为互联网的一等支付公民：HTTP 402 自动完成支付，不依赖 API Key 和人类预注册。

- 示例：Agent 自主调用付费 LLM、购买数据集、租用 GPU
- CAW 角色：让 Agent 持有和支配资金
- 02｜Trustless Agent Work Agreements

基于 ERC-8183 实现 Agent 间去信任工作协议：发布→托管→交付→验收/驳回→付款。

- 示例：Research Agent 发包给 Analyst Agent，自动比稿验收付款
- CAW 角色：Agent 资金账户，接收报酬、支付佣金、管理托管
- 03｜Agent Resource Procurement

Agent 根据任务自主发现、比价、采购算力、数据、API、存储等资源。

- 示例：Research Agent 买数据集；Trading Agent 订行情源
- CAW 角色：赋予 Agent 自主采购能力
- 04｜Autonomous Trading

Agent 自主执行链上交易策略：套利、做市、组合管理、流动性挖矿。

- CAW 角色：让 Agent 持有交易资金，授权边界内自主签署交易
- 05｜A2A Economy

多个 Agent 各自拥有钱包→Agent-to-Agent 经济体：互雇、拍卖、分账、管理 Treasury。

- CAW 角色：每个 Agent 独立 CAW 钱包，Agent 间直接支付、结算、分润

- **赛道规则**
- 项目必须围绕 Agent 与资金操作场景展开。
- Agent 的资金相关操作需通过 Cobo Agentic Wallet（CAW）完成。
- Agent 需要具备真实的资金执行能力，例如支付、转账、结算、收益管理或资产调度，而非仅停留在流程设计层面。
- 项目需体现 CAW 在 Agent 钱包管理、权限控制、安全隔离或自主支付等方面的价值。
- 最终成果需为可运行或可演示的产品原型（Demo），不接受纯 PPT、概念设计或 Mockup。

## 赛道2: z.ai 赛道｜web3 × long-horizon task

Z.AI 赛道关注 GLM-5.1 的长程任务能力：让 Agent 不只是回答问题，而是能自主拆解复杂任务、制定多步骤计划、持续调用工具、迭代修复，并完成从需求到交付的完整 Web3 工作流。参赛项目应体现长程、自主、持续执行，而不是只做一次性生成或普通 API 调用。

- **赛道建议方向（可选）**
- 01｜Agentic Dev Tools for Web3

让开发者用一句话，交付一个完整的 Web3 工程。重点不是代码补全，而是让 Agent 自主完成需求分析、合约设计、代码编写、测试迭代、漏洞修复、部署准备与文档输出的完整闭环。

- 自然语言驱动的 Web3 Agentic IDE：描述需求后，Agent 全程驱动合约开发、测试到部署。
- 智能合约审计 Agent：多轮自主分析、漏洞定位、修复建议，输出可追溯审计报告。
- 多链适配 Agent：将现有合约自主迁移并适配至目标链，处理兼容性差异。
- Web3 协议文档 Agent：读取协议文档，自主生成调用示例、测试代码与集成指引。
- 02｜AI-Powered 3D World Builder

不是“生成一个模型”，而是“建起一个世界”。Web3 的虚拟地产、NFT 展馆、3D 设计 / 空间构建、DAO 活动空间等都需要从空间规划、场景生成、资产配置到可部署包的完整生产链路。

- 虚拟地产与链上展馆：根据品牌调性和展览主题，自主规划空间布局、生成场馆结构、配置陈列方案，输出可部署的完整场景包。
- Web3 游戏场景与关卡：输入地图风格和玩法需求后，自主设计地形结构、功能分区、视觉风格，并持续迭代至符合游戏设计要求。
- 社区与活动空间：为 DAO 线上聚会、链上峰会、虚拟社区广场生成兼具功能性与辨识度的空间方案。
- 03｜AI × Creator Economy

让一个人 + 一个 Agent，完成过去需要整个创作团队才能完成的事。重点不是生成单张图或单段文字，而是从创意输入、资料收集、创作、排版、发布到链上变现的完整内容生产链路。

- 链上出版 Agent：自主收集资料、撰写、排版、发布研究报告 / 教程 / 白皮书，读者按需付费解锁并自动结算收益。
- IP 衍生内容 Agent：基于已有链上 IP，自主生成风格一致的衍生故事、动画脚本、周边资产。
- 动态 NFT 叙事引擎：根据链上事件、持仓变化、交易行为、时间周期或社区投票自动更新 NFT 视觉与元数据。
- 创作者 DAO 协作系统：多个 Agent 分工负责调研、写作、设计、发布、推广与自动分账。
